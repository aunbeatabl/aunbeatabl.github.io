I had my differential topology final yesterday; this was one of the problems: Compute the deRham cohomology groups of \(\mathbb{R}^2 - \{p, q\}\) for \(p, q\) distinct points.

I'm mainly writing this to practice using MathJax.

I used the Mayer-Vietoris sequence to compute this. Let \(U, V\) be open sets which cover the manifold, where \(U\) contains \(p\) but not \(q\) and \(V\) contains \(q\) but not \(p\), specifically so that their intersection is an open band. This is nice because both \(U\) and \(V\) are homotopic to \(S^1\) which we already know the cohomologies of: \[H^k_{dR}(S^n) = \begin{cases} \mathbb{R} \text{ if }k = 0, n\\ 0 \text{ else } \end{cases}\] So we have the following commutative diagram (turns out its really easy to make commutative diagrams in MathJax if you import it from q.uiver)
